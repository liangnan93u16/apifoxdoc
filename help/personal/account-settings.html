账号设置

在 Apifox 中，你可以通过账号设置来管理和自定义你的个人信息。

密码设置​

1.点击右上角头像，选择 "账号设置"。

2.在账号设置页面中，找到并点击 "密码设置" 部分。

3.输入和确认你输入的密码，点击保存，即成功设置了你的登录密码。

用户名修改​

1.在账号设置页面中，找到 "用户名" 部分，点击“修改"。

2.输入你想要设置的新用户名，点击"确认"，即可成功修改你的用户名。

手机/邮箱绑定​

1.在账号设置页面中，找到 "邮箱/手机号" 部分。

2.选择你要绑定的信息类型，点击“绑定”，即可绑定你的手机/邮箱。

tips:如果你需要解绑已绑定的手机号或邮箱，也可以在同一页面进行操作。

第三方账号绑定​

如果你是通过手机或邮箱注册并登录的，你可以将第三方账号与你的 Apifox 帐户关联起来。

1.在账号设置页面中，找到 "第三方账号绑定" 部分。

2.点击"绑定"，使用微信扫码后即可将你的微信账号与你的 Apifox 帐户进行关联。

更改团队昵称​

在团队中，你可以更改你的昵称，以便其他成员能够迅速识别你的身份。

1.进入你所在的团队，并找到"团队设置"。

2.在团队设置中，找到"我的团队昵称"，并点击"编辑"。

3.在编辑界面中，你可以修改你在团队中显示的昵称。

API 访问令牌管理​

你可以通过调用 Apifox 开放 API 来查看、修改和使用 Apifox 中的各类数据。

一、生成开放 API 访问令牌​

你需要生成一个可用的 访问令牌（Access Token） ，作为调用开放 API 的身份认证凭证，才可对开放 API 进行调用。

个人 API 访问令牌​

专属于你 Apifox 账号的访问令牌。该令牌跟你在 Apifox 中拥有的全部团队、项目的功能与数据权限完全一致。通过个人访问令牌调用开放 API 执行的各类操作等同于你账号在产品中执行的各类操作。

新建令牌

1.在账号设置页面中，找到 “API 访问令牌”。

2.点击“新建令牌”，填写名称并选择有效期，然后点击“保持并生成令牌”。

3.复制生成的令牌并妥善保存，因为一旦退出页面，你将无法再看到该令牌。

编辑/重置令牌

1.编辑令牌：当你需要修改一个已存在令牌的名称，或者修改令牌的时效性，可点击需要修改的令牌的"编辑"按钮，进行名称或有效期的修改。

tips:仅修改名称或有效期，不会重新生成令牌。

2.重置令牌：如果你忘记或丢失了令牌，可以在令牌的编辑弹窗中点击"重置 Token"，系统会生成一个新的令牌并提供给你。请记得复制并妥善保存新的令牌，原有令牌将立即失效。

二、使用令牌调用开放 API​

在调用开放 API 之前，你需要准备以下必填请求参数：生成的 Token（authorization）和要操作的项目ID（projectId）。

1.获取项目ID：进入项目设置，找到"基本设置"，然后在基本信息下复制"项目ID"。

2.准备好以上参数值后，进入 Apifox 开放 API 项目中，

tips：如果使用的是客户端，可在 API Hub 中找到 Apifox 开放 API，进入项目。

3.在项目中，进入"环境管理"，选择"正式环境"，将访问令牌和项目 ID 填写到指定的环境变量中，然后保存。

4.现在你可以根据 API 文档的说明，填写参数值并发送请求来调用具体的开放 API。

tips：你也可以在 Apifox 开放 API 文档中进行调试和请求，但是在项目中进行请求体验更佳。

账号注销​

1.在账号设置页面中，找到 "账号注销" 部分。

2.点击“注销”，会弹出一个身份证认证，获取验证码后点击“确定”，即可成功注销账号。

tip：当需要注销账号时，你需要先退出或解散所有团队后才能注销账号，可以在团队设置中解散团队。一旦成功注销账号，账号将无法再访问。
