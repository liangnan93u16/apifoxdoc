Alpha 版说明

Alpha 版为新功能尝鲜版，需要加官方微信群、QQ 群或钉钉群，才能参与 Alpha 版内测。

加群方式：点击此链接扫码加入。

  1. Apifox 新功能都会先在 Alpha 版上线，等稳定后才会合到正式版。
  2. Alpha 版可能会有一些 bug，如遇到问题及时在群里反馈，我们会第一时间解决。
  3. Alpha 版和正式版数据是互通的。
  4. 已经是 Alpha 版的，直接点击软件内更新。

Apifox 已规划或正在开发中的需求，可以查看 Road Map。
